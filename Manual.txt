시작 전 설정
1. 모든 채팅방명을 다르게 설정
2. 모든 채팅방의 내 대화명 같게 설정 ( 내 대화명이 다 다르다면 제외자에 설정하면 됨 )
3. 모든 채팅방의 공지사항을 다시 열지 않음 ( 공지사항이 있으면 오류발생 가능성이 있습니다 )

사용법
1. 카카오톡 / TalkManager 실행
2. 지연시간 설정 ( 1 / 1000 초) : 채팅방 갯수가 적을수록 시간을 높여 설정
   ( 컴퓨터의 무리가 갈 수 있음 / 최소 100 )
3. 내 대화명 : 내 대화명 입력
4. 채팅방 목록 설정 : 관리할 채팅방의 정확한 이름을 입력. (엔터로 구분)
   ( 채팅방을 실행하지 않아도 됩니다, 프로그램이 실행합니다 )
5-1. 고정문구 실행시간 설정 ( 분 ) : 고정문구를 실행할 시간 입력
5-2. 고정문구 설정 : 고정문구로 사용할 문장 입력
   ( 고정문구를 사용하지 않을 경우 ( 고정문구 실행 ) 버튼을 누르시면 비활성화 됩니다 )
6. 검색문구 설정 : 검색할 문구를 입력 ( 엔터로 구분 )
   ( 검색을 사용하지 않을 경우 ( 검색 실행 ) 버튼을 누르시면 비활성화 됩니다 )
7. 제외자 설정 : 모든 채팅방의 제외할 사람의 닉네임 입력 ( 엔터로 구분)
   ( 제외자를 사용하지 않을 경우 ( 제외자 실행 ) 버튼을 누르시면 비활성화 됩니다 )
   ( 방장, 부방장 이름을 적지 않을 경우 오류가 발생할 수 있습니다 )
8. 재시작 시간 설정 ( 분 ) : 채팅방을 재시작할 시간 설정
   ( 재시작 시간이 늘어날 수록 채팅방을 감시하는 시간이 늘어날 수 있습니다 )
9. 시작 버튼 클릭
10. 중지는 왼쪽 Shift 버튼 클릭
   ( 검색 중간에 마우스를 만지면 오류가 발생할 수 있습니다 )
